¿Qué es el Vault Transfer DS?

Un Design System es el lenguaje visual compartido entre diseño e ingeniería. En lugar de rediseñar cada botón o cada campo de formulario en cada pantalla, el DS define los componentes una sola vez y los reutiliza en toda la app.

Vault Transfer DS nació para resolver los flujos de una app de agenda de transferencias bancarias: cómo mostrar contactos, cómo confirmar una transferencia, cómo informar el estado. Cada componente responde a una necesidad real de esos flujos.
